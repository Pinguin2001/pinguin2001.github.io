What is PenguinStore?

PenguinStore is a new Apps and Games Store for W10M.

Which apps/games are here available?

In PenguinStore are lots of apps/games available made by the community.

I want to submit my app or game, how can I do that?

Send a message to Pinguin2001 on Telegram with the following content:
- Name of App or Game
- Short description
- Your developer name
- JPG/PNG file of your App Icon
- Category (e.g Socials)
- APPX/APPXBundle file
- Dependencies
