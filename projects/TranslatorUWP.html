TranslatorUWP



Download (Version 3.1)

Dependencies

arm32

arm64

x64

x86

Features:

- Translate text (powered by Lingva translate backend)
- Import text from file
- Import text from voice
- Save translate text to file
- Spanning on Windows 10X
- Continuum (on Windows 10 Mobile based devices)

System requirements:

- arm32, arm64, x86 or x86_64 based Windows device
- OS Build: Windows 10 Build 10.0.16229.0 (1709) or higher, Windows 10 Mobile Build 10.0.15063.0 (1703) or higher
- 512mb ram, 1gb ram recommended
- Internet connection
